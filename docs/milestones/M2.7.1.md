# M2.7.1 — Void Feedback Composition Test Hotfix

## Status

Validated locally — build and full test suite passed.

## Scope

This hotfix corrects one contradictory expected value in the M2.7 void-feedback composition test.

The test inputs are:

- `void/a`: measured 30%, reference 20%, coefficient `+1 pcm/%void` → `+10 pcm`;
- `void/b`: measured 10%, reference 20%, coefficient `+2 pcm/%void` → `-20 pcm`.

Therefore the composed void reactivity is `-10 pcm`. The previous test expected `+30 pcm`, which ignored the signed deviation of `void/b`.

No production code, API, numerical method, physical coefficient semantics, canonical ordering or runtime behavior changed.
