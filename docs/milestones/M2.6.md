Validated locally — build and all tests passed; baseline carried into M2.7.

## Scope

- strongly typed signed temperature-reactivity coefficient;
- explicit fuel/coolant temperature feedback definitions;
- linear reference-temperature feedback law;
- canonical multi-feedback evaluation and normal reactivity-breakdown composition;

## Acceptance criteria

- reference temperature produces zero feedback;
- signed coefficients behave correctly above and below reference;
- fuel and coolant categories remain distinct diagnostics;
- duplicate contribution IDs fail fast;
- generic solver contains no plant-specific coefficient constants;
- full fixed-step feedback loop is independent of external pulse segmentation;
- existing architecture and wall-clock guards remain green.
