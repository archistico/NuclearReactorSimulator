# M2.3 — Neutron Kinetics

## Goal

Introduce deterministic point-reactor kinetics with delayed-neutron groups so total reactivity evolves a normalized neutron population without yet coupling directly to thermal power.

## Scope

- strongly validated delayed-neutron fractions, decay constants and normalized populations;
- immutable delayed-group definitions and point-kinetics parameter sets;
- critical-equilibrium state initialization;
- standard point-kinetics ODE system with arbitrary delayed-neutron group count;
- deterministic bounded RK4 internal substepping inside the fixed runtime timestep;
- prompt-critical margin and parameter-relative dollars/cents diagnostics;
- instantaneous logarithmic growth rate and signed reactor-period diagnostics;
- runtime pulse-segmentation determinism tests;
- explicit numerical fail-fast boundary.

## Non-goals

- plant-specific RBMK kinetic constants;
- neutron population to thermal-power conversion;
- decay heat;
- thermal/void/xenon feedback implementation;
- spatial diffusion or channel-wise neutron fields.

## Acceptance

- zero-reactivity critical equilibrium remains stable within numerical tolerance;
- positive sub-prompt reactivity increases neutron population;
- negative reactivity decreases neutron population;
- delayed-neutron group state evolves explicitly;
- prompt-critical threshold is derived from the active `β_eff`;
- identical fixed-step histories remain deterministic regardless of external pulse segmentation;
- invalid parameter/state coverage fails fast;
- build and complete test suite pass locally.
