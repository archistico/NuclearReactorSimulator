# M3.3 — Aggregated Core-Zone Model

Status: **VALIDATED**

## Goal

Introduce a configurable spatial core structure while preserving the validated global point-kinetics model.

## Delivered

- `CoreZoneCoordinate` with no fixed grid-size assumption.
- `CoreZonePowerFraction`.
- `CoreZoneDefinition` referencing canonical fuel, structure and coolant plant domains.
- `AggregatedCoreDefinition` with canonical ordering, unique ids/coordinates, eager plant-reference validation and normalized nominal power shares.
- `CoreZoneState` and `AggregatedCoreState` with exact zone-set validation and normalized current power shares.
- `AggregatedCorePowerSolver` for deterministic global-to-local fission-power partitioning.
- Exact total-power closure through deterministic residual allocation.
- `CoreZoneSnapshot` / `AggregatedCoreSnapshot` with local committed-state thermal/hydraulic diagnostics and void projection.
- Tests for configurable non-3x3 topology, canonical ordering, invalid references, fraction closure, dynamic power shares and deterministic allocation.

## Deliberately deferred

- Fuel-channel groups: M3.4.
- Hydraulic core paths: M3.4/M3.5.
- Steam drums: M3.6.
- Spatial neutron diffusion or independent per-zone kinetics: future fidelity work.
