# M2.8 — Iodine/Xenon Dynamics

## Goal

Close the M2 reactor-physics foundation with a deterministic, history-dependent I-135/Xe-135 poisoning model that feeds the validated compositional reactivity boundary.

## Included

- normalized immutable I-135 and Xe-135 inventories;
- configurable I/Xe production rates at reference fission power;
- configurable iodine and xenon decay constants;
- neutron-population-dependent xenon burnup;
- analytic finite-step coupled inventory evolution;
- long-operation equilibrium initialization;
- signed configurable xenon reactivity coefficient;
- named `Xenon` reactivity contribution;
- diagnostic production/decay/burnup rates;
- deterministic runtime and pulse-segmentation integration tests.

## Excluded

- plant-specific RBMK poison constants;
- spatial xenon oscillations;
- absolute isotope-number-density depletion;
- samarium and other poisons;
- burnup-dependent yields/cross sections.

## Acceptance

- build succeeds with warnings as errors;
- all existing and new tests pass;
- equilibrium state remains steady under matching constant inputs;
- shutdown can produce xenon buildup from iodine feeding;
- higher neutron population increases xenon burnup;
- xenon contributes through `ReactivityContributionKind.Xenon` only;
- identical logical fixed-step history produces identical final state independent of external pulse segmentation.

## Status

VALIDATED — local build and full test suite passed. M2 reactor-physics foundation is closed.
