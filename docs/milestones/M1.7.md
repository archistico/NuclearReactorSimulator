# M1.7 — Simplified Water/Steam Phase Model

## Goal

Close M1 by providing the first deterministic production thermodynamic model that resolves a fluid node's conserved mass, fixed volume and conserved internal energy into pressure, temperature and coarse water/steam phase.

## Scope

- explicit fluid-phase classification;
- strongly validated saturated-mixture vapor quality;
- IAPWS-IF97 Region-4 saturation-pressure reference boundary;
- simplified saturated liquid/vapor density and internal-energy correlations;
- deterministic two-phase quality/temperature closure;
- simplified subcooled/compressed liquid closure;
- simplified superheated-vapor closure;
- fail-fast unsupported-state envelope;
- integration through the existing `IFluidThermodynamicModel` seam;
- deterministic repeated fixed-step verification;
- documentation of fidelity limits and replacement path.

## Acceptance criteria

- [x] existing two-argument `FluidThermodynamicState` construction remains source compatible;
- [x] phase is explicit in production thermodynamic states;
- [x] vapor quality exists only for saturated mixtures;
- [x] saturation pressure near 100 °C is consistent with the Region-4 reference equation;
- [x] a constructed saturated mixture round-trips temperature, pressure and quality;
- [x] dense liquid states resolve as subcooled/compressed liquid;
- [x] low-density high-energy states resolve as superheated vapor;
- [x] unsupported states fail explicitly rather than producing non-finite values;
- [x] conserved mass/internal energy remain owned by `FluidNodeInventory`;
- [x] repeated fixed-step closure remains deterministic;
- [x] no hydraulic, valve, pump or heat-transfer solver duplicates thermodynamic logic;
- [x] model is explicitly documented as educational and not full IAPWS-IF97.

## Out of scope

- complete IAPWS-IF97 regions/backward equations;
- supercritical-water modelling;
- metastable steam;
- detailed enthalpy/entropy property APIs;
- viscosity/thermal conductivity/surface tension;
- two-phase slip, void-distribution and critical-flow correlations;
- reactor neutronics and void-reactivity feedback (M2).
