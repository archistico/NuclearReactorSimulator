## Fail-closed evidence rule

Unavailable/missing metrics never pass a reference target.

Reference validation uses immutable presentation evidence or explicit canonical solver outputs only. It does not reach through Application into private solver/audit state.

## Sensitivity rule

Sensitivity probes compare explicit baseline and perturbed canonical runs. The analysis layer reports response direction/magnitude and normalized sensitivity but never mutates model parameters itself.

## GUI/App pre-M10 hardening

## Deliberate non-goals

- licensing-grade or externally certified validation;
- automatic calibration/parameter fitting;
- invented historical reference values;
- a second physics owner;
- pixel-diff screenshot testing;
- M10 operator-computer or supervisory automation implementation.
