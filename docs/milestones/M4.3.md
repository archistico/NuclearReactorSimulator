# M4.3 — Condenser, Vacuum & Hotwell

## Status

Validated.

M4.2 was the validated baseline used to build M4.3. Local build and the complete automated test suite for M4.3 were subsequently reported successful; M4.3 is now the validated baseline for M4.4.

## Scope delivered

- canonical `CondenserSystemDefinition` over the validated M4.2 turbine expansion topology;
- exactly one condenser per turbine stage group/exhaust seam;
- canonical steam-space and hotwell fluid-node ownership;
- replaceable one-to-one cooling-water/environment boundary definitions;
- deterministic committed-state condensation limited by condenser capacity, vapor inventory and available heat rejection;
- conservative steam-space-to-hotwell mass transfer;
- explicit external heat-rejection power accounting;
- dynamic condenser pressure/vacuum diagnostics resolved from canonical fluid inventory;
- hotwell mass/temperature/phase diagnostics;
- backward-compatible M4.2 supplemental source-term composition preserving exactly one thermofluid integration;
- topology, cooling-limit, conservation, vacuum-response, determinism and input-coverage tests;
- ADR 0034 and `docs/CONDENSER_VACUUM_HOTWELL.md`.

## Explicitly deferred

- condensate and feedwater pumps;
- feedwater heaters/deaeration;
- replacement of the simplified M3 feedwater source;
- detailed condenser tube/cooling-water hydraulics;
- non-condensable gases, ejectors and vacuum pumps;
- automatic condenser/hotwell control;
- generator/grid coupling.

These belong to M4.4 and later roadmap milestones.
